Sun/Pacman


This is a assignment that I did in my other MTEC class. It look simple but the coding is a bit complicated. This was a good project to show because although it was a bit of a struggle to get this done, it was a good learning experience and it was intersting to learn how to code interactive animations like this.


You interact with it by putting your mouse over the sun and pacman will appear. To move the back man you need to use the arrow keys to move it left or right.
